# Get-PowerPlanAlias

## SYNOPSIS
Get Power Plan aliases

## SYNTAX

```
Get-PowerPlanAlias [<CommonParameters>]
```

## DESCRIPTION
Returns all power plan settings, including name, type and parent/child association

## EXAMPLES

### EXAMPLE 1
```
Get-PowerPlanAlias
```
